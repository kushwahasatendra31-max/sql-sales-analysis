# SQL Sales Analysis Project

## Overview

This project demonstrates SQL-based business data analysis using a sales dataset. The objective is to analyze sales performance, identify top customers, track revenue trends, and generate business insights using SQL queries.

## Database Schema

The sales table contains:

- Order ID
- Order Date
- Customer Name
- Product
- Category
- Quantity
- Sales Amount
- City

## SQL Concepts Used

- SELECT Statements
- Aggregations (SUM, COUNT)
- GROUP BY
- ORDER BY
- Subqueries
- Window Functions
- Ranking Functions
- Data Analysis

## Business Questions Solved

1. What is the total sales revenue?
2. Who are the top customers?
3. Which product categories generate the highest revenue?
4. What are the monthly sales trends?
5. How do customers rank based on sales amount?
